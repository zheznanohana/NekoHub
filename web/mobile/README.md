# 📱 NekoHub 移动版使用指南

## 🎉 欢迎使用 NekoHub！

NekoHub 是一款现代化的任务管理和消息处理平台，支持 Web、Android 多平台。

---

## 📥 安装指南

### Android 安装

1. **下载 APK**
   - 从 Telegram 下载附件
   - 或从 GitHub Releases 下载

2. **允许安装**
   ```
   设置 → 安全 → 安装未知来源应用 → 允许
   ```

3. **安装应用**
   - 点击 APK 文件
   - 点击"安装"
   - 等待安装完成

4. **打开应用**
   - 点击"打开"
   - 输入账号密码登录

## 🌙 深色模式

### 自动切换
- 跟随系统设置
- 自动检测 prefers-color-scheme

### 手动切换
- 设置 → 外观 → 深色模式

### 深色模式配色
- 背景：#1a1a1a
- 卡片：#1e1e1e
- 文字：#ffffff
- 次要文字：#b0b0b0

## 🔧 技术细节

### 技术栈
- **前端：** Vue 3.4 + Vite 5 + Element Plus
- **移动端：** Capacitor 5.x
- **后端：** Flask + SQLAlchemy
- **实时：** WebSocket (Socket.IO)
- **数据库：** SQLite

### 系统要求
- **Android:** 8.0+ (API 26+)
- **内存:** 2GB+
- **存储:** 100MB+

### 权限说明
- **网络:** 访问 API 接口
- **通知:** 接收推送消息
- **存储:** 缓存数据 (可选)

## ❓ 常见问题

### Q: 安装失败怎么办？
A: 检查是否允许"安装未知来源应用"，重启手机后重试。

### Q: 无法登录？
A: 检查网络连接，确认账号密码正确，联系管理员。

### Q: 消息不更新？
A: 下拉刷新，或检查 WebSocket 连接状态。

### Q: 深色模式不生效？
A: 检查系统设置，或在应用内手动切换。

### Q: 如何反馈问题？
A: 通过 Telegram 联系开发者。
